# Open Graph Images

This directory contains language-specific Open Graph images (1200x630px) for social media sharing.

## Required Images

Each language requires an OG image named: `{language}-og-image.jpg`

Required files:
- `en-og-image.jpg` - English
- `es-og-image.jpg` - Spanish (Español)
- `pt-og-image.jpg` - Portuguese (Português)
- `de-og-image.jpg` - German (Deutsch)
- `fr-og-image.jpg` - French (Français)
- `he-og-image.jpg` - Hebrew (עברית)
- `ru-og-image.jpg` - Russian (Русский)
- `uk-og-image.jpg` - Ukrainian (Українська)

## Image Specifications

- **Dimensions**: 1200 × 630 pixels (exact)
- **Format**: JPG (optimized for web)
- **File size**: < 1MB per image
- **Content**: 
  - Main headline in respective language
  - Logo/branding
  - Visual elements representing Polish citizenship/legal services
  - High contrast for readability

## Design Guidelines

1. **Safe Zone**: Keep important text/elements within central 1200×600px area
2. **Text**: Large, bold, readable at thumbnail size
3. **Brand Colors**: Use design system colors (primary/secondary)
4. **Mobile Preview**: Test how it looks when shared on mobile devices
5. **Consistency**: Maintain visual consistency across all language versions

## Generation Tools

- Canva: https://www.canva.com/create/og-images/
- Figma templates
- Adobe Photoshop/Illustrator
- Online OG image generators

## Testing

Test OG images using:
- Facebook Sharing Debugger: https://developers.facebook.com/tools/debug/
- Twitter Card Validator: https://cards-dev.twitter.com/validator
- LinkedIn Post Inspector: https://www.linkedin.com/post-inspector/
